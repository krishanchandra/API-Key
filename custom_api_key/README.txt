This module provide API Key & Node JSON output.

1. Enable this module 
2. Goto admin/config/system/site-information.
3. Add a API key in text field and save form.
4. Now we can access node content in JSON format base on API Key Authentication.
5 For access node content in JSON format we need to access url with two argument
For Example - Our Saved key is hgfgfhgfhgfh4646789ablk 
We want access node which have NID 6
Then We need to acces url www.exaplesite.com/json/hgfgfhgfhgfh4646789ablk/6
